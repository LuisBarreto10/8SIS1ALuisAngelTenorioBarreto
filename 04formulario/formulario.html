Todos los formularos que se manerjan dentro de la web utilizan de envio para cada uno de los datos, que nos sirve para las solicitudesde informacion(request) los mas comunes son:
GET: este metodo se encarga de enviar la informacion por la url o la salida estandar del navegador, esto propicia se pueda visualizar toda la informacion del fortmulario de la url
POST: este metodo se encarga de enviar la informacion por la url; pero de forma oculta, lo que ofrece cierto de seguridad, este metodo se usa comunmente para insertar un nuevo registro dentro de las sistemas de informacion
